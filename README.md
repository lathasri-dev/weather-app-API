# 🌦️ Weather App (Flask-Based)

## 📌 Overview
This is a web-based weather application built using Flask that provides real-time weather information for any city. The app fetches live data from the OpenWeatherMap API and displays key weather details in a simple and user-friendly interface.

## 🚀 Features
- 🌍 Search weather by city name
- 🌡️ Real-time temperature (in Celsius)
- ☁️ Weather condition description
- 🎯 Dynamic weather icons from API
- ⚠️ Error handling for invalid city input
- 🔄 Interactive form-based user input

## 🛠️ Technologies Used
- Python
- Flask (Backend Framework)
- HTML (Frontend Template)
- OpenWeatherMap API
- Requests Library

## ⚙️ How It Works
1. User enters a city name in the input field.
2. Flask handles the request and sends an API call to OpenWeatherMap.
3. The API returns weather data in JSON format.
4. The app processes the data and displays:
   - City Name
   - Temperature
   - Weather Description
   - Weather Icon
